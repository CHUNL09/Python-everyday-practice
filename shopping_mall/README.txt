Python package "shopping_mall"

Purpose: 购物商城（提供用户注册、登录、购买、结算等功能）

Author: Chun Liu (cedrela_liu@163.com)

Blog: http://www.cnblogs.com/cedrelaliu/

Licence: no

Version: 1.0 -- 2016-01-2

Versions of Python supported: 3.x 

External modules required: none

Files in the package:
   account.txt -- 用于存储用户帐户信息，格式：用户名,密码,帐户余额
   shopping_mall.py  -- 主程序文件
   shopping_mall_flow_chart -- 流程图，用于描述程序的逻辑

特别说明：
1 充值模块输入n结束充值，否则在一次充值过后还会询问是否充值。
2 选购商品时需要输入商品名称
3 测试用账号可以新注册，也可以从account.txt文件中挑选

Function details:
1 用户注册--新用户输入'r'后进行注册 ;
2 用户登录验证--老用户键入'l'到登录界面，输入用户名密码登录;
3 充值功能--可以让用户进行充值;
4 购物--用户可以选购商品，选购时需要输入具体商品类别和名称;
5 购物车结算--在购物车结算时打印用户购物车内商品，评估帐户余额是否购买；这里提供
  充值模块，可以让用户充值后结算；也提供删除商品功能，可以删除购物车中的商品.
